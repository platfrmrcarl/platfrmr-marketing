# Specification: ADK Agent System

## Requirements
- Rewrite the legacy Python agents in TypeScript.
- Utilize the `@google/adk` framework for agent communication and tool execution.
- Use `GoogleSearchTool` for research.
- Implement a custom tool for LinkedIn API publishing.

## Tech Stack
- **@google/adk**: Multi-agent framework.
- **Gemini Models**: AI intelligence.
- **TypeScript**: Implementation language.
