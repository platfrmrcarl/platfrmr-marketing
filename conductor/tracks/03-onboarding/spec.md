# Specification: User Onboarding

## Requirements
- Collect `target_niche`, `target_audience`, `target_topics`, and `preferred_hashtags`.
- Split the process into multiple logical steps.
- Ensure only authenticated users can access the onboarding page.
- Persist data to the user's Firestore document.

## Tech Stack
- **React**: UI components.
- **Firebase Firestore**: Data persistence.
- **Next.js**: Routing and authentication middleware/hooks.
